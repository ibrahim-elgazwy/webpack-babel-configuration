## Usage

Webpack Babel starter that support:
 * New versions of Javascript (es6, ...)
 * Support CSS, SASS Loaders

### Installation

Install the dependencies

```sh
$ npm install
```

### Build
Compile and build

```sh
$ npm run build
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm run dev
```
